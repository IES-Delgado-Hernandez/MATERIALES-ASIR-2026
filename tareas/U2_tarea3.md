# Tarea RA2.3: Fases del proyecto y planificación temporal

## Objetivo
Definir de forma clara y estructurada las **fases del proyecto**, las **actividades necesarias para su desarrollo** y una **planificación temporal realista**, que sirva como base para la ejecución del Proyecto Intermodular.

---

## Criterios de evaluación
- **c)** Se han identificado las fases del proyecto especificando su contenido y plazos de ejecución.  
- **e)** Se han determinado las actividades necesarias para el desarrollo del proyecto.

---

## Tareas a realizar

### 1. Identificación de las fases del proyecto
Define las fases principales de tu proyecto, alineadas con su desarrollo técnico.  
Como referencia orientativa (puedes adaptarlas a tu caso):

- Análisis
- Diseño
- Implementación / Configuración
- Pruebas y validación
- Documentación
- Entrega final

Para cada fase indica brevemente:
- Qué se va a realizar.
- Qué resultado o entregable se espera.

---

### 2. Actividades del proyecto
Para cada fase, identifica las **actividades necesarias** para llevarla a cabo.  
Las actividades deben estar directamente relacionadas con el trabajo real que realizarás en tu proyecto.

Ejemplo:
- Instalación de servicios.
- Configuración de sistemas.
- Diseño de esquemas o diagramas.
- Pruebas de funcionamiento.
- Elaboración de documentación técnica.

---

### 3. Planificación temporal
Elabora una **planificación temporal** del proyecto indicando:
- Duración aproximada de cada fase.
- Orden de ejecución.
- Dependencias básicas entre fases o actividades.

Debes utilizar como mínimo un **diagrama de Gantt** y si lo consideras oportuno, otro formato de planificación que domines.

La planificación debe ser **realista y coherente con el calendario del módulo**.

---

## Desarrollo del proyecto
El contenido de esta tarea deberá integrarse directamente en la documentación del proyecto.  
La planificación definida servirá como referencia para el desarrollo real del proyecto en las siguientes semanas.

No se requiere ninguna entrega adicional distinta al documento indicado.

---

## Formato y entrega
- Documento en **formato PDF** con nombre:  
  **TareaRA2_3_Nombre completo del alumno.pdf**
- El documento deberá incluir:
  - Portada.
  - Encabezado y numeración de páginas.
  - Fases, actividades y planificación claramente estructuradas.

---

La entrega se realizará a través de **Moodle**, dentro del plazo establecido.
