Asegúrate de tener Docker instalado y sigue estos pasos:# 1. Clona el cuartel general
git clone [https://github.com/tu-usuario/shield-sentinel.git](https://github.com/tu-usuario/shield-sentinel.git)

# 2. Entra en la base de operaciones
cd shield-sentinel

# 3. Configura tus variables de entorno (Usa el archivo de ejemplo)
cp .env.example .env

# 4. Levanta los escudos
docker-compose up -d
¡Listo! Accede a tu panel de control en http://localhost:3000.📖 Documentación¿Quieres profundizar en los manuales tácticos?
